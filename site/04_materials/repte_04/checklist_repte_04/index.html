Checklist del Repte 4¶

Ús de la checklist¶

Esta checklist servix per revisar si el Repte 4 ja ha convertit el backend estructurat del curs en una API publicable, provada i documentada, prou clara per ser consumida després en un flux híbrid.

Endpoints principals implementats¶

  • [ ] existixen endpoints prioritaris alineats amb el domini del projecte
  • [ ] hi ha almenys un endpoint principal de consulta
  • [ ] hi ha almenys un endpoint principal d'escriptura o acció
  • [ ] els endpoints protegits requerixen autenticació quan toca

Validacions i codis d'estat coherents¶

  • [ ] les dades d'entrada es validen de manera coherent
  • [ ] els codis d'estat són consistents amb el resultat de l'operació
  • [ ] el contracte d'entrada i eixida és recognoscible entre endpoints similars

Tractament d'errors¶

  • [ ] hi ha respostes clares per errors de validació
  • [ ] hi ha respostes clares per accessos no autoritzats o no autenticats
  • [ ] hi ha comportament coherent per recursos inexistents o errors previsibles

Documentació d'API¶

  • [ ] l'API està documentada amb endpoints, mètodes i camps principals
  • [ ] la documentació descriu respostes correctes i errors rellevants
  • [ ] les instruccions de prova són suficients per a un altre membre de l'equip

Proves executades¶

  • [ ] existix una col·lecció mínima o equivalent de proves API
  • [ ] s'han executat casos correctes i casos d'error
  • [ ] hi ha evidència de consum real o d'ús extern controlat de l'API

README actualitzat¶

  • [ ] el README o documentació tècnica reflectix com provar o consumir l'API
  • [ ] la documentació està alineada amb l'estat real del backend

AI log si s'ha usat IA¶

  • [ ] existix AI log o registre equivalent si s'ha utilitzat IA
  • [ ] l'ús de IA queda validat amb proves, execució o revisió tècnica

Preparació per al Repte 5¶

  • [ ] l'API actual es pot reutilitzar en integració híbrida sense reestructuració crítica
  • [ ] estan identificats els endpoints que seran més útils per a consum extern o automatització
  • [ ] la documentació i les proves actuals deixen una base clara per al consum del Repte 5
