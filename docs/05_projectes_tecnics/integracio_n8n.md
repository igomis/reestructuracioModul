# Integració de n8n

## Ubicació recomanada

Repte 5, amb connexió secundària al Repte 4.

## Cas d’ús suggerit

Automatització d’un procés de negoci:

- webhook d’entrada
- consulta a servei extern
- transformació de dades
- notificació o registre
- resposta al backend

## Evidències

- workflow exportat
- prova del webhook
- integració amb el backend
- documentació del flux
