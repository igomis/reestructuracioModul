# Projecte backend base del curs

## Objectiu
Definir un producte backend únic o una família de productes compatibles amb els reptes del mòdul.

## Possibles dominis
- gestió d’incidències
- reserves
- biblioteca tècnica
- inventari
- gestió interna de centre

## Components mínims
- autenticació
- persistència
- API
- documentació
- proves
