# Projecte backend base del curs

## Finalitat del document
Delimitar línies de treball futures per al bloc de projectes tècnics del mòdul, connectades amb el producte backend del curs i amb valor formatiu real.

## Nota de treball futur
En la fase de projectes tècnics es valorarà una possible línia vinculada a **"tokenitzar alguna cosa"**, sempre en clau backend, verificable i didàctica. Esta nota no defineix encara l’arquitectura final del projecte.

## Exemples breus d’enfocaments didàctics
- emissió i validació de tokens d’accés o invitació per a fluxos d’alta/control d’entrada
- gestió de credencials signades per a integració segura entre serveis
- sistema simplificat de traçabilitat d’actius digitals amb tokens interns i registre d’operacions
