# Sistema d’evidències

## Evidències comunes

- repositori individual
- historial de commits o traça equivalent
- documentació operativa
- proves o verificacions reproduïbles
- `AI log` quan toque
- defensa tècnica individual

## Criteri general

El pes principal recau en allò que l’alumnat pot explicar, validar i defensar.

La cooperació pot aportar context útil:

- anàlisi inicial compartida
- comparació de tecnologies
- revisió creuada
- checkpoints comuns
- contrast final de resultats

Però mai substituïx:

- autoria individual
- traçabilitat individual
- evidències individuals
- defensa individual

## Lectura metodològica per fases

- en `R2` la base comuna en `PHP` facilita control de mínims sobre dades, lògica, estat i autenticació
- des de `R3` el mateix projecte pot continuar amb frameworks diferents
- el canvi de framework no altera el tipus d'evidència exigida
