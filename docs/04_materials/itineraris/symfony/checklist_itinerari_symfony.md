# Checklist de l'itinerari Symfony

## Revisió mínima

- [ ] el mateix projecte de `R2` s'ha traslladat a `Symfony` sense perdre el flux funcional
- [ ] el `README` explica com arrancar el projecte
- [ ] controladors, serveis i persistència tenen rols recognoscibles
- [ ] l'autenticació està lligada a operacions reals del domini
- [ ] hi ha proves o verificacions bàsiques reproduïbles
- [ ] la base queda preparada per publicar `API`

## Recordatori metodològic

- `Symfony` entra a partir de `R3`
- el repositori, les evidències i la defensa continuen sent individuals
- el framework no substituïx la comprensió del flux
