# Professorat

## Què conté este espai

Esta secció funciona com a porta d'entrada curta per a professorat. No reescriu tot el paquet docent: selecciona els documents que convé obrir segons el moment del curs i els tradueix a ús operatiu.

El criteri és simple:

- accés ràpid per preparar, conduir, avaluar i ajustar el curs
- derivació cap a la documentació llarga quan cal detall o contrast
- separació clara respecte del camí d'[alumnat](../alumnat/index.md)

## Per on començar

Si has d'obrir el curs o reprendre'l ràpidament, l'ordre recomanat és este:

1. [Com preparar el curs](com_preparar_el_curs.md)
2. [Com conduir els reptes](com_conduir_els_reptes.md)
3. [Com avaluar i fer seguiment](com_avaluar_i_fer_seguiment.md)

Si ja tens el curs en marxa, entra directament en:

- [Com gestionar ritmes i suport](com_gestionar_ritmes_i_suport.md)
- [Com pilotar i ajustar](com_pilotar_i_ajustar.md)
- [Materials de classe compartibles](../08_materials_compartibles/index.md)
