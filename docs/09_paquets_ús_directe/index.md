# Paquets d'ús directe

## Finalitat del bloc
Este espai agrupa els materials més útils segons el moment del curs perquè el professorat no haja d'anar saltant entre massa pàgines. No crea contingut nou llarg: reutilitza i ordena el que ja existix.

## Quan convé usar-lo
- si vols preparar una sessió o tram del curs amb molt pocs clics
- si necessites saber què obrir tu i què donar a l'alumnat
- si vols tindre clar què projectar, què usar per checkpoint i què reservar per a defensa o tancament

## Criteri d'ús
- el bloc [Materials compartibles](../08_materials_compartibles/index.md) continua sent la base de peces curtes
- estos paquets són una capa superior d'ordenació per moment docent
- si necessites aprofundir, cada paquet et deriva a apunts reals, materials de sessió i portals d'ús
