# Criteris de treball

## Convencions
- Un document per decisió important
- Una issue per tasca concreta
- Noms descriptius i consistents
- Revisar coherència curricular abans de tancar una tasca

## Flux recomanat
1. Crear issue
2. Crear o modificar document
3. Revisar coherència amb RA, reptes i avaluació
4. Fer commit amb missatge clar
